# MinLength

A modifier that creates a validation rule to ensure that a given string is longer or equal in length to the given value.

## Schemas

- `string`: [Learn more](/schemas/string).

## Arguments

- `min`: The number which the input value should be more or equal to.
- `message` (Optional): A custom object with errors messages to be displayed when validation fails or there are type errors.

## Example

```ts
import { minLength, string } from 'validathor';

const minLengthRule = string([minLength(4)]);

try {
  const parsedValue = minLengthRule.parse('Hello');
  console.log('Parsed value:', parsedValue);
} catch (error) {
  console.error('Parsing failed:', error.message);
}
```
