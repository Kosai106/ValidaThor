# MaxLength

A modifier that creates a validation rule to ensure that a given string is shorter or equal in length to the given value.

## Schemas

- `string`: [Learn more](/schemas/string)

## Arguments

- `max` The number which the input value should be less or equal to.
- `message` (Optional): The error message to throw if the value is invalid.

## Example

```ts
import { maxLength, string } from 'validathor';

const maxLengthRule = string([maxLength(4)]);

try {
  const parsedValue = maxLengthRule.parse('Hej');
  console.log('Parsed value:', parsedValue);
} catch (error) {
  console.error('Parsing failed:', error.message);
}
```
