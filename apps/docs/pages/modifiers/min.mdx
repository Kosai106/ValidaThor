# Min

A modifier that creates a validation rule to ensure that a given numeric value meets the minimum requirement.

## Schemas

- `number`: [Learn more](/schemas/number).

## Arguments

- `min`: The minimum value that the input value must meet or exceed.
- `message` (Optional): A custom object with errors messages to be displayed when validation fails or there are type errors.

## Example

```ts
import { min, number } from 'validathor';

const minRule = number([min(10)]);

try {
  const parsedValue = minRule.parse(42);
  console.log('Parsed value:', parsedValue);
} catch (error) {
  console.error('Parsing failed:', error.message);
}
```
