# MaxDate

A modifier that creates a validation rule to ensure that a given date is less or equal to the input date.

## Schemas

- `date`: [Learn more](/schemas/date).

## Arguments

- `date`: The date which the input value should be less or equal to.
- `message` (Optional): A custom object with errors messages to be displayed when validation fails or there are type errors.

## Example

```ts
import { maxDate, date } from 'validathor';

const maxDateRule = date([maxDate(new Date('2023-06-14'))]);

try {
  const parsedValue = maxDateRule.parse(new Date('2023-04-20'));
  console.log('Parsed value:', parsedValue);
} catch (error) {
  console.error('Parsing failed:', error.message);
}
```
