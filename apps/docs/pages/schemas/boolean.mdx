# Boolean

A function that creates a parsing rule to ensure that a given value is a boolean.

## Arguments

1. (Optional): A custom error message to be displayed when the parsing fails. If not provided, the default message `Expected a Date` will be used.


## Example

```ts
import { boolean } from 'validathor';

const booleanRule = boolean('Value must be a boolean');

try {
  const parsedValue = booleanRule.parse(true);
  console.log('Parsed value:', parsedValue);
} catch (error) {
  console.error('Parsing failed:', error.message);
}
```
