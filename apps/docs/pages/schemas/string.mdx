# String

A function that creates a parsing rule to ensure that a given value is a string.

## Arguments

1. (Optional): A list of modifiers
   - `minLength`: [Learn more](/modifiers/min-length)
   - `maxLength`: [Learn more](/modifiers/max-length)
   - `email`: [Learn more](/modifiers/email)
2. (Optional): A custom error message to be displayed when the parsing fails. If not provided, the default message `Expected a string` will be used.

## Example

```ts
import { string } from 'validathor';

const stringRule = string(undefined, 'Value must be a boolean');

try {
  const parsedValue = stringRule.parse('Hello world');
  console.log('Parsed value:', parsedValue);
} catch (error) {
  console.error('Parsing failed:', error.message);
}
```
