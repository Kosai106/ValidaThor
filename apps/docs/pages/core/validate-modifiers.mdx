# ValidateModifiers

<Callout type="info" emoji="ℹ️">
  This function is for *primarily* internal use.
</Callout>

Some schemas accept modifiers, such as `min()` or `email()` which can modify the way a schema works.

`validateModifiers()` is a function used within these schemas to run the modifiers' validations.

## Example

```ts
const myNumberModifiers = [min(2), max(12)];

ValidateModifiers(4, myModifiers);
```
